🚢 Maritime Logistics Dashboard (DataDNA Challenge)
This project is my entry for the Onyx Data March–April 2026 Challenge. I used Power BI to look at shipping data and find ways to make cargo move faster.

🚀 The Main Goal
The goal of this challenge was to find ways to reduce cargo movement time by 15%.

🔍 Top 3 Insights
The Suez Effect: I tracked how the 2021 Suez Canal blockage still affects shipping times today.
The Aging Fleet: I found that Legacy ships (25+ years old) are slower and cause a 1.05-day delay compared to newer ships.
The Bottleneck: Terminal 460 in AMER is the slowest spot, making it the best place to start making improvements.
